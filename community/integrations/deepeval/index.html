使用 DeepEval 进行 LLM/RAG 单元测试#

DeepEval 为 AI 智能体和基于 LLM 的应用程序提供单元测试功能。它为 LlamaIndex 用户提供了极其简单的接口来测试 LLM 输出，帮助开发者在生产环境中捕捉破坏性变更。

DeepEval 提供了一套标准化框架来衡量响应质量，且完全开源。

安装与设置#

安装 DeepEval 非常简单且无需额外配置：

pip install -U deepeval
# 可选步骤：登录后可获得测试仪表板功能！
deepeval login

安装完成后，可创建 test_rag.py 开始编写测试用例。

test_rag.py
import pytest
from deepeval import assert_test
from deepeval.metrics import AnswerRelevancyMetric
from deepeval.test_case import LLMTestCase


def test_case():
    answer_relevancy_metric = AnswerRelevancyMetric(threshold=0.5)
    test_case = LLMTestCase(
        input="如果鞋子不合脚怎么办？",
        # 请替换为你的LLM应用实际输出
        actual_output="我们提供30天无理由全额退款服务。",
        retrieval_context=[
            "所有客户均可享受30天无理由全额退款服务。"
        ],
    )
    assert_test(test_case, [answer_relevancy_metric])

运行测试命令如下：

deepeval test run test_rag.py

若已登录，可在 deepeval 仪表板分析评估结果：

评估指标类型#

DeepEval 为 RAG 应用提供了一套标准化的单元测试框架。它将评估分解为测试用例，并提供多种可自由组合的评估指标：

  • G-Eval
  • 摘要质量
  • 答案相关性
  • 忠实度
  • 上下文召回率
  • 上下文精确度
  • 上下文相关性
  • RAGAS
  • 幻觉检测
  • 偏见检测
  • 毒性检测

DeepEval 持续将最新研究成果融入评估指标。点击此处 了解完整指标列表及计算方法。

为 LlamaIndex 应用评估 RAG#

DeepEval 与 LlamaIndex 的 BaseEvaluator 类完美集成。以下示例展示了如何将 DeepEval 评估指标以 LlamaIndex 评估器形式使用。

from llama_index.core import VectorStoreIndex, SimpleDirectoryReader
from deepeval.integrations.llama_index import DeepEvalAnswerRelevancyEvaluator

# 详情参考LlamaIndex快速入门
documents = SimpleDirectoryReader("你的数据目录").load_data()
index = VectorStoreIndex.from_documents(documents)
rag_application = index.as_query_engine()

# 输入示例
user_input = "什么是LlamaIndex？"

# LlamaIndex返回包含输出字符串和检索节点的响应对象
response_object = rag_application.query(user_input)

evaluator = DeepEvalAnswerRelevancyEvaluator()

评估执行方式：

evaluation_result = evaluator.evaluate_response(
    query=user_input, response=response_object
)
print(evaluation_result)

完整评估器列表#

可通过以下方式导入全部6个评估器：

from deepeval.integrations.llama_index import (
    DeepEvalAnswerRelevancyEvaluator,
    DeepEvalFaithfulnessEvaluator,
    DeepEvalContextualRelevancyEvaluator,
    DeepEvalSummarizationEvaluator,
    DeepEvalBiasEvaluator,
    DeepEvalToxicityEvaluator,
)
