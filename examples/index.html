示例#

LlamaIndex 提供了丰富的示例集合，展示多样化的使用场景、集成方案和功能特性。本页面重点介绍关键示例，助您快速入门。

在左侧导航栏中，您还可以找到许多示例笔记本，展示各类 llama-index 组件和使用案例的实际应用。

利用 LlamaIndex 的智能体功能构建强大的 AI 助手：

  • 函数调用智能体 - 学习函数调用智能体和 AgentWorkflow 的基础知识
  • ReAct 智能体 - 使用 ReAct（推理与行动）模式构建智能体
  • 代码执行智能体 - 能够编写和执行代码的智能体
  • 多智能体工作流 - 使用 AgentWorkflow 构建多智能体协作流程
