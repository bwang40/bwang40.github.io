使用 VoyageAI 多模态嵌入实现多模态检索¶

VoyageAI 发布了多模态嵌入模型，本笔记本将演示如何利用 VoyageAI 多模态嵌入实现多模态检索。

演示步骤如下：

  1. 从 HuggingFace 下载包含图像的数据集。
  2. 使用 VoyageAI 多模态嵌入为图像构建多模态索引。
  3. 通过多模态检索器同时检索与查询相关的图像。

环境安装¶

我们将使用 VoyageAI 多模态嵌入进行检索，并采用 MongoDB 作为向量数据库。

def plot_images(images, image_indexes):
    images_shown = 0
    for image_ndx in image_indexes:
        image = Image.open(BytesIO(images[image_ndx]))

        plt.subplot(8, 8, images_shown + 1)
        plt.imshow(image)
        plt.xticks([])
        plt.yticks([])

        images_shown += 1
        if images_shown >= 50:
            break

    plt.tight_layout()
    plt.show()
