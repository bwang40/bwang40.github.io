从 ServiceContext 迁移至 Settings 配置#

自 v0.10.0 版本起，我们引入了全新的全局 Settings 对象以取代旧的 ServiceContext 配置方案。

新的 Settings 对象采用全局设置模式，其参数采用延迟初始化机制。诸如 LLM 或嵌入模型等属性，仅当底层模块实际需要时才会被加载。

以往使用服务上下文时，许多模块往往不会真正调用它，同时还强制在运行时将所有组件加载至内存（即使这些组件并未被使用）。

配置全局设置意味着您将修改 LlamaIndex 中所有模块的默认行为。例如，如果您不使用 OpenAI，配置示例如下：

from llama_index.llms.ollama import Ollama
from llama_index.embeddings.huggingface import HuggingFaceEmbedding
from llama_index.core import Settings

Settings.llm = Ollama(model="llama2", request_timeout=120.0)
Settings.embed_model = HuggingFaceEmbedding(
    model_name="BAAI/bge-small-en-v1.5"
)

通过此设置，您可以确保框架永远不会调用 OpenAI 服务。

Settings 对象支持与旧版 ServiceConext 几乎相同的所有属性。完整列表可查阅文档页面。

完整迁移指南#

以下是从 ServiceContext 完全迁移至 Settings 的示例：

迁移前

from llama_index.embeddings.openai import OpenAIEmbedding
from llama_index.core.node_parser import SentenceSplitter
from llama_index.llms.openai import OpenAI
from llama_index.core import ServiceContext, set_global_service_context

service_context = ServiceContext.from_defaults(
    llm=OpenAI(model="gpt-3.5-turbo"),
    embed_model=OpenAIEmbedding(model="text-embedding-3-small"),
    node_parser=SentenceSplitter(chunk_size=512, chunk_overlap=20),
    num_output=512,
    context_window=3900,
)
set_global_service_context(service_context)

迁移后

from llama_index.embeddings.openai import OpenAIEmbedding
from llama_index.core.node_parser import SentenceSplitter
from llama_index.llms.openai import OpenAI
from llama_index.core import Settings

Settings.llm = OpenAI(model="gpt-3.5-turbo")
Settings.embed_model = OpenAIEmbedding(model="text-embedding-3-small")
Settings.node_parser = SentenceSplitter(chunk_size=512, chunk_overlap=20)
Settings.num_output = 512
Settings.context_window = 3900

局部配置#

上述内容涉及全局配置。若需按模块配置，所有模块接口均已更新为接受所使用对象的 kwargs 参数。

如果您使用 IDE，智能提示应会自动填充 kwargs 参数，但这里仍提供一些示例：

# 向量存储索引仅需嵌入模型
index = VectorStoreIndex.from_documents(
    documents, embed_model=embed_model, transformations=transformations
)

# ... 直到创建查询引擎时才需要LLM
query_engine = index.as_query_engine(llm=llm)

# 文档摘要索引在构造时同时需要LLM和嵌入模型
index = DocumentSummaryIndex.from_documents(
    documents, embed_model=embed_model, llm=llm
)
