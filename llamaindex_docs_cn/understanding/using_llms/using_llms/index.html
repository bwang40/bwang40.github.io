使用大语言模型#

Tip

如需查看我们支持的大语言模型列表及其功能对比，请参阅我们的大语言模型模块指南。

构建基于大语言模型的应用时，首要步骤之一就是选择使用哪种大语言模型；它们各有所长且价格各异，您可能需要同时使用多个模型。

LlamaIndex 为众多不同的大语言模型提供了统一接口。使用大语言模型可以像安装相应集成包一样简单：

pip install llama-index-llms-openai

然后通过一行代码调用：

from llama_index.llms.openai import OpenAI

response = OpenAI().complete("William Shakespeare is ")
print(response)

注意这需要您在环境变量中设置名为 OPENAI_API_KEY 的API密钥；更多详情请参阅入门教程。

complete 方法也有异步版本 acomplete。

您还可以通过调用 stream_complete 获取流式响应，该方法会返回一个生成器，实时生成产生的词元：

handle = OpenAI().stream_complete("William Shakespeare is ")

for token in handle:
    print(token.delta, end="", flush=True)

stream_complete 方法也有异步版本 astream_complete。

聊天接口#

LLM 类还实现了 chat 方法，支持更复杂的交互：

messages = [
    ChatMessage(role="system", content="You are a helpful assistant."),
    ChatMessage(role="user", content="Tell me a joke."),
]
chat_response = llm.chat(messages)

stream_chat 和 astream_chat 方法同样可用。

指定模型#

许多大语言模型集成支持多个模型。您可以通过向 LLM 构造函数传递 model 参数来指定模型：

llm = OpenAI(model="gpt-4o-mini")
response = llm.complete("Who is Laurie Voss?")
print(response)

多模态大语言模型#

部分大语言模型支持多模态聊天消息。这意味着您可以传入文本与其他模态（图像、音频、视频等）的混合内容，大语言模型将能处理。

目前，LlamaIndex 通过内容块支持在 ChatMessages 中使用文本、图像和音频。

from llama_index.core.llms import ChatMessage, TextBlock, ImageBlock
from llama_index.llms.openai import OpenAI

llm = OpenAI(model="gpt-4o")

messages = [
    ChatMessage(
        role="user",
        blocks=[
            ImageBlock(path="image.png"),
            TextBlock(text="Describe the image in a few sentences."),
        ],
    )
]

resp = llm.chat(messages)
print(resp.message.content)

工具调用#

部分大语言模型（OpenAI、Anthropic、Gemini、Ollama等）直接通过API调用支持工具调用——这意味着无需特定提示和解析机制即可调用工具和函数。

from llama_index.core.tools import FunctionTool
from llama_index.llms.openai import OpenAI


def generate_song(name: str, artist: str) -> Song:
    """Generates a song with provided name and artist."""
    return {"name": name, "artist": artist}


tool = FunctionTool.from_defaults(fn=generate_song)

llm = OpenAI(model="gpt-4o")
response = llm.predict_and_call(
    [tool],
    "Pick a random song for me",
)
print(str(response))

关于更高级的工具调用细节，请参阅使用OpenAI的深度指南。相同方法适用于所有支持工具/函数的LLM（如Anthropic、Gemini、Ollama等）。

您可以在工具指南中了解更多关于工具和代理的信息。

可用的大语言模型#

我们支持与OpenAI、Anthropic、Mistral、DeepSeek、Hugging Face等数十种模型的集成。查看我们的大语言模型模块指南获取完整列表，包括如何运行本地模型。

Tip

关于隐私和大语言模型使用的一般说明可在隐私页面找到。

使用本地大语言模型#

LlamaIndex不仅支持托管的大语言模型API；您还可以在本地运行如Meta的Llama 3等本地模型。例如，如果您已安装并运行Ollama：

from llama_index.llms.ollama import Ollama

llm = Ollama(
    model="llama3.3",
    request_timeout=60.0,
    # 手动设置上下文窗口以限制内存使用
    context_window=8000,
)

更多关于使用和配置大语言模型的细节，请参阅自定义大语言模型操作指南。
